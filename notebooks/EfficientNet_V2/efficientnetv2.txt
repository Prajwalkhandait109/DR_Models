      DIABETIC RETINOPATHY DETECTION USING DEEP LEARNING

Architecture Name: EfficientNetV2 (efficientnetv2_rw_s)

--------------------------------------------------------------------------------
1. OVERVIEW
--------------------------------------------------------------------------------
Diabetic Retinopathy (DR) is a diabetes-related retinal disorder that damages the
blood vessels of the retina and may lead to permanent blindness if not diagnosed
at an early stage. Automated retinal image analysis using deep learning can help
ophthalmologists perform faster and more reliable screening.

This project presents an automated diabetic retinopathy grading system using
EfficientNetV2 — a modern convolutional neural network architecture optimized
for faster training, improved parameter efficiency, and high classification
accuracy. The model classifies retinal fundus images into five severity grades:
No_DR, Mild, Moderate, Severe, and Proliferate_DR.

Transfer learning was applied using ImageNet-pretrained EfficientNetV2 weights.
The model was fine-tuned on retinal fundus images using a single-phase end-to-end
training strategy. GradCAM explainability analysis was performed on the final
convolution layer to visualize the retinal regions influencing model predictions,
supporting medical interpretability and trustworthy AI.

--------------------------------------------------------------------------------
2. DATASET DESCRIPTION
--------------------------------------------------------------------------------
Source      : Kaggle — "Diabetic Retinopathy 224x224 2019 Data"
              (sovitrath/diabetic-retinopathy-224x224-2019-data)
Downloaded  : Via KaggleHub API
Image Folder: colored_images/

Classes (5-class grading scale):
  0 — No_DR            (No diabetic retinopathy)
  1 — Mild             (Early stage retinal abnormalities)
  2 — Moderate         (Visible hemorrhages and exudates)
  3 — Severe           (Advanced retinal damage)
  4 — Proliferate_DR   (Neovascularization and severe retinal pathology)

Dataset Statistics:
  Total Images : 3,662

Dataset Split:
  Training Set   : 70%
  Validation Set : 15%
  Testing Set    : 15%

DataLoader:
  Batch Size  : 16
  Shuffle     : True (train), False (validation/test)

All images are pre-resized to 224×224 pixels and organized into separate folders
according to their diabetic retinopathy severity grades.

--------------------------------------------------------------------------------
3. PREPROCESSING AND AUGMENTATION
--------------------------------------------------------------------------------
Training Transforms:
  - Resize(224×224)
  - RandomHorizontalFlip(p=0.5)
  - RandomRotation(±10°)
  - ToTensor()

Validation/Test Transforms:
  - Resize(224×224)
  - ToTensor()

Data Augmentation Purpose:
  - Improves model generalization
  - Reduces overfitting
  - Simulates retinal orientation variations
  - Helps model learn robust retinal features

Image Size:
  224 × 224 × 3 (RGB)

--------------------------------------------------------------------------------
4. MODEL ARCHITECTURE DESCRIPTION
--------------------------------------------------------------------------------
Base Model : EfficientNetV2 Small — efficientnetv2_rw_s
Source     : timm (PyTorch Image Models)
Pretrained : ImageNet pretrained weights

EfficientNetV2 is a convolutional neural network architecture designed for:
  - Faster training
  - Better parameter efficiency
  - Reduced computational complexity
  - Improved feature extraction performance

Key Architectural Components:
  - Convolution Layers
  - MBConv Blocks
  - Fused MBConv Layers
  - Squeeze-and-Excitation Attention
  - Compound Scaling Strategy

Architecture Flow:
  Input Image (224×224×3)
        |
  [Convolution Stem]
        |
  [Fused MBConv Blocks]
        |
  [MBConv Blocks]
        |
  [Feature Extraction Layers]
        |
  [Global Average Pooling]
        |
  [Fully Connected Classification Head]
        |
  [Output Layer → 5 DR Classes]

Transfer Learning:
  ImageNet-pretrained weights were used to initialize the EfficientNetV2
  backbone. The final classification layer was modified to classify retinal
  images into five diabetic retinopathy severity classes.

Target Layer for GradCAM:
  - model.conv_head

Output:
  5-class logits → argmax for final prediction

5. TRAINING STRATEGY
--------------------------------------------------------------------------------
Single-Phase Fine-Tuning:

  The pretrained EfficientNetV2 model was fine-tuned end-to-end using retinal
  fundus images. Validation monitoring and early stopping were used to prevent
  overfitting.

  ── Hyperparameters ──────────────────────────────────────────────────────────
  Optimizer         : Adam
  Learning Rate     : 1e-4
  Loss Function     : CrossEntropyLoss
  Batch Size        : 16
  Epochs            : 10
  Early Stopping    : Patience = 3

  ── Additional Techniques ───────────────────────────────────────────────────
  Transfer Learning : Enabled
  Validation Loss Monitoring : Enabled
  GPU Acceleration  : CUDA (if available)

  ── Training Workflow ───────────────────────────────────────────────────────
  1. Dataset loading
  2. Image preprocessing
  3. Data augmentation
  4. EfficientNetV2 fine-tuning
  5. Validation monitoring
  6. Early stopping
  7. Model evaluation
  8. GradCAM explainability analysis

6. EVALUATION METRICS
--------------------------------------------------------------------------------

  6.1 ACCURACY
  ─────────────────────────────────────────────────────────────────────────────
  Accuracy measures the percentage of correctly classified retinal images.

      Accuracy = Correct Predictions / Total Predictions

  Higher accuracy indicates better diabetic retinopathy classification
  performance.

  6.2 CONFUSION MATRIX
  ─────────────────────────────────────────────────────────────────────────────
  A confusion matrix was generated to analyze:
    - Correct classifications
    - Incorrect classifications
    - Class-wise prediction performance

  Rows    : Actual classes
  Columns : Predicted classes

  The confusion matrix helps identify visually confusing diabetic retinopathy
  stages and class imbalance effects.

  Output:
    outputs/confusion_matrix.png

  6.3 ROC-AUC CURVES
  ─────────────────────────────────────────────────────────────────────────────
  One-vs-Rest ROC-AUC curves were generated for all five diabetic retinopathy
  classes using softmax probabilities.

  ROC-AUC measures:
    - class separability
    - prediction confidence
    - diagnostic capability

  Higher AUC values indicate better classification performance.

  Output:
    outputs/roc_auc.png

  6.4 CLASSIFICATION REPORT
  ─────────────────────────────────────────────────────────────────────────────
  The classification report includes:
    - Precision
    - Recall
    - F1-score
    - Support

  These metrics provide detailed class-wise performance analysis for all diabetic
  retinopathy severity levels.

  Output:
    outputs/classification_report.txt

  6.5 GRADCAM EXPLAINABILITY
  ─────────────────────────────────────────────────────────────────────────────
  Gradient-weighted Class Activation Mapping (GradCAM) was used to visualize
  the retinal regions influencing model predictions.

  Target Layer:
    model.conv_head

  GradCAM overlays heatmaps on retinal images to show:
    - lesion-focused attention
    - retinal abnormality regions
    - model decision areas

  Two sets of GradCAM outputs:
    Correct Predictions:
      GradCAM visualizations for correctly classified retinal images

    Wrong Predictions:
      GradCAM visualizations for misclassified retinal images

  Output Folder:
    outputs/gradcam/
      ├── correct/
      └── wrong/
